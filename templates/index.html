Drift Lens is an unsupervised drift detection technique for deep learning models on unstructured data. It is based on distribution distances of the embedding representations of deep learning models. The tool is able to perform real-time drift detection and characterization by indicating i) when, ii) how , and iii) where drift occurs. Specifically, it is able to detect in which data windows drift occur, the severity and pattern, and which labels have been mostly affected.

You can experiment with drift detection on:

  • Pre-uploaded use cases: Where drift can be controlled with several patterns and intensities on pre-uploaded models and data. (Run Controlled Drift Experiment)
  • Your own data: Where you can upload your data to verify if drift is present in new data streams. (Run Your Drift Experiment)
